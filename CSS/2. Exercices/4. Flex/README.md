# Exercice 4 - FlexBox
Cet exercice a pour but de vous familliariser avec le concept de positionnement d'élément avec FlexBox.

_Outils nécessaires_
* Git
* Notepad++
* Un fureteur

_Connaissances acquises_
1. Positionner un élément par rapport à un autre
2. Aligner les éléments
3. Justifier les éléments

## À faire
1. Mettre les 3 carrés sur une ligne.
2. Centrer les 3 carrés sur une ligne.
3. Center les 3 carrés horizontalement et verticalement.
4. Mettre les carrés sur une ligne pour qu'ils occupent toute l'espace.
