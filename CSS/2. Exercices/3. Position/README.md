# Exercice 3 - Position absolute vs relative
Cet exercice a pour but de vous familliariser avec le concept de positionnement.

_Outils nécessaires_
* Git
* Notepad++
* Un fureteur

_Connaissances acquises_
1. Positionner un élément par rapport à un autre
2. Principe de positionnement relatif
3. top, left, right & bottom

## À faire
1. Mettre le carré rouge en haut à droite du carré gris pâle.
2. Mettre le carré rouge en haut à droite du carré gris foncé.
3. Mettre le carré rouge en bas à droite du carré gris foncé.
4. Mettre le carré rouge en bas à droite du carré gris pâle.
5. Centrer le carré rouge dans le carré gris pâle.
6. Centrer le carré rouge dans le carré gris foncé.
