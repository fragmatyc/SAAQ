Exercie 4

Les variables

  1. Ouvrez la console développeur avec F12
  2. Ouvrez l'onglet console pour voir les logs.
  3. Éditez index.js pour :
    • Créer une fonction logique() acceptant 1 paramètre
    • Appliquer le duck typing pour valider que le paramètre est un tableau.
    • Créer un tableau temporaire pour stocker des valeurs pour retourner.
    • Pour chaque élément du tableau, appliquer la logique suivante, avec un switch :
      • Si la valeur est 1, ajouter 'A' dans le tableau temporaire.
      • Si la valeur est 'A', ajouter 1 dans le tableau temporaire.
      • Pour toute autre valeur, ajouter '*' dans le tableau temporaire.
      • Si le paramètre n'est pas un tableau, retourner null.
    • Retourner le tableau temporaire.
